📘 **b3046 - R2 (15줄)**  
두 수 R1, S가 주어졌을 때 R2를 구하는 문제.  
- `S = (R1 + R2) / 2` 공식을 변형하여 계산  
- 단순 수학 공식 문제  

---

📘 **b3047 - ABC (30줄)**

- 배열 정렬(`Arrays.sort`)

📘 **b3058 - 짝수를 찾아라 (28줄)**

- 짝수 판정(`% 2 == 0`)  
- 합과 최솟값 동시 갱신  

---

📘 **b3059 - 등장하지 않는 문자의 합 (28줄)**  
문자열에서 등장하지 않은 알파벳의 아스키 코드 합을 출력하는 문제.  
- boolean 배열로 알파벳 등장 체크

📘 **b3449 - 해밍 거리 (25줄)**

- 단순 구현 문제  

---

📘 **b3460 - 이진수 (26줄)**
