## 📘 b1000 - A+B
- 두 정수를 입력받아 더한 결과를 출력하는 문제
- 입력: `1 2`
- 출력: `3`

---

## 📘 b1001 - A-B
- 두 정수를 입력받아 첫 번째 수에서 두 번째 수를 뺀 결과를 출력하는 문제
- 입력: `3 2`
- 출력: `1`
